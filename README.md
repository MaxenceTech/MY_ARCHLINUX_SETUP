# MY_ARCHLINUX_SETUP

Ce dépôt contient des scripts automatisés pour gérer l'installation d'Arch Linux sur ma Surface Laptop GO.

## Usage :
```
./install.sh
```
Puis après le redémarrage, 
```
cd /archinstall/SCRIPT
./secure-boot.sh
```
Puis après le redémarrage, 
```
cd /archinstall/SCRIPT
./secondscript.sh
```
